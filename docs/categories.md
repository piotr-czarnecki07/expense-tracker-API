# categories

Categories are represented by numbers

```json
{
    "1": "Groceries",
    "2": "Leisure",
    "3": "Electronics",
    "4": "Utilities",
    "5": "Clothing",
    "6": "Health",
    "7": "School",
    "8": "Home",
    "9": "Other"
}
```

When listing the categories that expense should be marked with, provide NUMBERS instead of text.\
It is not necessary to enclose numbers within `""`.
